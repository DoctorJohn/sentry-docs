Once the feature is enabled on the SDK and the SDK is initialized, you can send logs using the `sentry_sdk.logger` APIs.

The `logger` namespace exposes six methods that you can use to log messages at different log levels: `trace`, `debug`, `info`, `warning`, `error`, and `fatal`.

```python
from sentry_sdk import logger as sentry_logger

sentry_logger.trace('Starting database connection')
sentry_logger.debug('Cache miss for user')
sentry_logger.info('Updated global cache')
sentry_logger.warning('Rate limit reached for endpoint')
sentry_logger.error('Failed to process payment')
sentry_logger.fatal('Database connection pool exhausted')
```

You can also pass additional attributes directly to the logging functions, avoiding the need to format strings manually. These properties will be sent to Sentry, and can be searched from within the Logs UI, and even added to the Logs views as a dedicated column.

```python
from sentry_sdk import logger as sentry_logger

sentry_logger.trace('Starting database connection', database="users")
sentry_logger.debug('Cache miss for user', user_id=123)
sentry_logger.info('Updated profile', profile_id=345)
sentry_logger.warning('Rate limit reached for endpoint', endpoint='/api/results/', is_enterprise=False)
sentry_logger.error('Failed to process payment', order_id="or_2342", amount=99.99)
sentry_logger.fatal('Database connection pool exhausted', database="users", active_connections=100)
```

For more advanced use cases, you can pass structured attributes using the `attributes` parameter. This allows you to set custom attribute names without automatic prefixing:

```python
from sentry_sdk import logger as sentry_logger

sentry_logger.error(
    'Payment processing failed', 
    user_id=12345,
    attributes={
        'payment.provider': 'stripe',
        'payment.method': 'credit_card',
        'payment.currency': 'USD',
        'user.subscription_tier': 'premium'
    }
)

sentry_logger.warning(
    'Database query slow',
    attributes={
        'db.operation': 'SELECT',
        'db.table': 'users',
        'db.duration_ms': 1500,
        'service.instance': 'web-01'
    }
)
```
